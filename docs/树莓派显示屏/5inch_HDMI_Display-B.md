测试页面1

:::note
本显示屏兼容树莓派所有版本（4代/3代/2代/1代），但树莓派5代需额外升级驱动。
:::

:::tip 省电小技巧
使用背光开关关闭显示屏背光时，功耗可降低约60%，适合电池供电场景。
:::

:::warning 触摸功能限制
仅用作显示时无需占用树莓派IO资源，但启用触摸功能会占用GPIO口，可能和其他外设冲突。
:::

:::danger 禁止操作123
请勿同时连接USB供电接口和树莓派母座取电，否则会导致电压过载，损坏显示屏或树莓派。
:::
